# 🏦 Python ATM Simulator

A simple command-line ATM simulation built using Python.
This project demonstrates core programming concepts by mimicking real-world ATM operations like balance checking, deposits, withdrawals, and transaction tracking.

---

## 🚀 Features

* 🔐 Secure PIN authentication
* 💰 Check account balance
* ➕ Deposit money
* ➖ Withdraw money with balance validation
* 🔄 Change PIN
* 📜 Transaction history with summary
* ⚙️ Functional programming using `map()`, `filter()`, and `lambda`

---

## 🧠 Concepts Used

This project is built using fundamental Python concepts:

* 🔁 Loops (for continuous ATM menu)
* 🔀 Conditional statements (if-elif-else)
* 📥 Input / 📤 Output handling
* 🔢 Arithmetic operations
* 🧩 Functions (modular design)
* 📦 Dictionaries (account data storage)
* 🎯 Input validation (PIN and amount checks)
* 📝 String formatting (f-strings)
* ⚙️ Functional programming:

  * `map()`
  * `filter()`
  * `lambda`
